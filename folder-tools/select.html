1 Скачать по ссылке Ссылка на плагин: Code => Download ZIP => choices.min.js choices.min.css и подключить


2 Прописать код в JS-файле: 

const element = document.querySelector('.js-choice');
const choices = new Choices(element);

или с добавлением функции поиска
const element = document.querySelector('.js-choice');
const choices = new Choices(element, {
  searchEnabled: true; // добавление строки поиска
});
