SLICK

Ссылка на плагин

1 Подключить плагины jQuery

  <script type="text/javascript" src="//code.jquery.com/jquery-1.11.0.min.js"></script>
  <script type="text/javascript" src="//code.jquery.com/jquery-migrate-1.2.1.min.js"></script>


2 Скачать по ссылке архив.

3 Перенести из архива файлы slick.css slick.min.js и подключить.


4 Создать HTML-разметку 

  <div class="your-class">
    <div>your content</div>
    <div>your content</div>
    <div>your content</div>
  </div>

5 Прописать скрипт в JS 

  'use strict'
  $(document).ready(function () {
    $('.your-class').slick({
      arrows: false, // стрелки для переключения картинки
      fade: true,
      autoplay: 10000, // время смены картинки
      dots: true // подключение точек для переключения картинки
    });
  });

ДОПОЛНЕНИЕ: CSS-стили

  • 1 Главной секции задать:
    position: relative;
  • 2 Слайдер-блоку необходимо задать, чтобы слайдер не вылез за пределы видимой части:
    position: absolute;
z-index: -1;
left: 0;
top: 0;
right: 0;
bottom: 0;
  • 3 Слайдер-ячейкам прописываем:
    height: 100vh;
background-repeat: no-repeat;
background-size: cover;
background-position: center;
  • 4 Точкам для переключения также прописываем стили:
    .slick__slider .slick-dots {
  margin: -55px auto 0;
  padding: 0 15px; // зависит от размеров проекта
  position: relative;
  z-index: 2;
  max-width: 1230px; // свои параметры задаём
}
  • 5 Кнопке точек задаём:
    .slick__slider .slick-dots button {
  width: 15px;
  height: 15px;
  border-radius: 50%;
  background-color: #fff;
  border: none;
  padding: 0;
  font-size: 0;
}
  • 6 Стили кнопки точек в момент активности точки:
    .slick__slider .slick-dots .slick-active button {
background-color: yellowgreen;
cursor: pointer;
}
  • 7 Чтобы выстроить в ряд точки задаём:
    .slick__slider .slick-dots li {
display: inline-block;
}
  • 8 И отступы между точками
    .slick__slider .slick-dots li:not(:last-child) {
margin: 0 20px 0 0;
}

ИНИЦИАЛИЗАЦИЯ


HTML


<!-- контейнер слайдера -->
<div class="swiper-container">
  <!-- враппер -->
  <div class="swiper-wrapper">
    <!-- слайды -->
    <div class="swiper-slide">Slide 1</div>
    <div class="swiper-slide">Slide 2</div>
    <div class="swiper-slide">Slide 3</div>

  </div>
  <!-- кнопки пагинации -->
  <div class="swiper-pagination"></div>

  <!-- кнопки для навигации -->
  <div class="swiper-button-prev"></div>
  <div class="swiper-button-next"></div>
</div>


Подключение файлов CSS 

<link rel="stylesheet" href="https://unpkg.com/swiper/swiper-bundle.min.css"/>

ВАЖНО подключить стили раньше основных файлов



Подключение файлов JS 

<script src="https://unpkg.com/swiper/swiper-bundle.min.js"></script>

ВАЖНО подключить скрипты раньше основных файлов



Далее в родном JS-файле необходимо прописать следующий код для последующей настройки слайдера: 

const swiper = new Swiper('.swiper-container', {
  // кол-во слайдов для показа
  slidesPerView: 1,


  // кол-во перелистываемых слайдов
  slidesPerGroup: 3,


  // бесконечный цикл прокрутки
  loop: true,


  // автопрокрутка
  autoplay: {
    // Пауза между прокруткой (миллисекунды)
    delay: 1000,

    // Закончить на последнем слайде
    stopOnLastSlide: true,

    // Отключить после ручного переключения
    disableOnInteraction: false,
  },


  // эффект переключения слайдов сменой прозрачности
  effect:  “fade”,

  // Дополнение к fade
  fadeEffect: {
    // параллельная смена прозрачности
    crossFade: true,
  },


  // пагинация (ТОЧКИ)
  pagination: {
    el: '.swiper-pagination',
    clickable: true,
  },


  // навигация (КНОПКИ)
  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },
});

XV ВИРТУАЛЬНЫЕ СЛАЙДЫ


Виртуальные слайдеры, чтобы не заполнять html, можно создать через js, прописав нужное кол-во. В данном случае 500 слайдов. 

JS

// фомирование слайдера
virtual: {
    slides: (function () {
      let slide = []
      for (let I = 0; I < 500; i++) {
        slide.push(`<div class=”swiper-container__text”>Слайд №${i}</div>`);
      }
      return slide;
    }()),
  },
});
